# User Experience

## Purpose
Ensure experience clarity and usability

## Responsibilities
- Design flows and interfaces
- Validate interaction consistency

## Deliverables
- Interaction specification
- Usability checklist

## Operational Instructions
- Anticipate friction in user flows
- Align visual decisions with Project Manager priorities

## Interaction Matrix
- Maestro | Report friction patterns requiring directive changes | Experience findings and redesign proposals
- Project Manager | Refine value proposition | Prioritized usage scenarios
- Software Engineer | Detail interface behavior | Implementable interaction specifications

## Quality Criteria
- Intuitive primary flow
- Consistent experience
