# Value Streams

This document groups Maestro features by value stream.

## 1. Developer Velocity
Outcome: faster delivery with coordinated agent execution.

Capabilities:
- Multi-agent Observe/Think/Act runtime.
- Persona orchestration and handoff matrix.
- TUI command center and creation wizards.

## 2. Governance and Compliance
Outcome: safer and auditable AI-assisted engineering.

Capabilities:
- Markdown governance validation for scopes/personas/skills.
- Strict conventions for error handling and observability.
- Readiness checks and release quality gates.

## 3. User Onboarding and Adoption
Outcome: lower onboarding friction and faster first value.

Capabilities:
- User and project onboarding flows.
- Resume state machine and skip/continue controls.
- Practical guides and smoke-test playbooks.

## 4. Operations and Extensibility
Outcome: predictable runtime operations and platform growth.

Capabilities:
- Pluggable provider registry and adapters.
- CLI lifecycle commands for setup, run, and diagnostics.
- Packaging pipelines for Debian, macOS, and Omarchy.

## Feature-to-Stream Mapping
- Runtime orchestration -> Developer Velocity.
- Quality gates and conventions -> Governance and Compliance.
- Onboarding state machine -> User Onboarding and Adoption.
- Provider registry and packaging -> Operations and Extensibility.
