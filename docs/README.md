# Maestro Documentation Hub

This directory is organized into four required areas:

1. [Maestro Execution Plans](Maestro_Execution_Plans/README.md): Development execution plans, milestones, and release candidates.
2. [Practical Guides](Practical_Guides/README.md): Hands-on guides for onboarding and operational checks.
3. [User Manual](User_Manual/README.md): Command and panel reference for operators.
4. [Maestro Manifesto](Maestro_Manifesto/README.md): Philosophy, architecture doctrine, conventions, feature levels, and value streams.

If you are new to the project, start with the manifesto, then follow the practical guides.
